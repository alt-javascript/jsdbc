# S01: Teradata driver implementation — UAT

**Milestone:** M002
**Written:** 2026-03-30T19:28:50.254Z

## UAT — Teradata Driver\n\n**Requires:** Live Teradata 16.20+ instance, teradatasql binary installed for your platform.\n\n### Setup\n\nEdit `packages/teradata/test/TeradataDriver.spec.js` to match your instance:\n```js\nurl: 'jsdbc:teradata://your-host:1025/yourdatabase',\nusername: 'youruser',\npassword: 'yourpassword',\n```\n\n### Run\n\n```bash\nnpm test -w packages/teradata\n```\n\n### Expected\n\nAll compliance suite tests pass:\n- Connection: creates and closes a connection\n- Statement DDL/DML: INSERT, SELECT, UPDATE, DELETE\n- PreparedStatement: parameterised INSERT and SELECT, NULL handling\n- ResultSet: cursor iteration, getRows(), getColumnNames(), 1-based index access, empty result set\n- Transactions: commit persists, rollback discards\n- DriverManager: resolves connection via URL\n\n### Smoke test (manual)\n\n```js\nimport '@alt-javascript/jsdbc-teradata';\nimport { DataSource } from '@alt-javascript/jsdbc-core';\n\nconst ds = new DataSource({ url: 'jsdbc:teradata://host/db', username: 'u', password: 'p' });\nconst conn = await ds.getConnection();\nconst stmt = await conn.createStatement();\nconst rs = await stmt.executeQuery('SELECT 1 AS n');\nrs.next();\nconsole.log(rs.getInt('n')); // 1\nrs.close(); await stmt.close(); await conn.close();\n```
